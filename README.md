# はじめに

このガイドは、Gatsby.jsを使ったブログのスターターテンプレートをカスタマイズするための手順を説明するためのものです。

## スタートガイド

### 必要なツールのインストール

まずはじめに、お使いのPCにNode.jsがインストールされている必要があります。
Node.jsは、JavaScriptを使ってサーバーサイドのプログラムを開発するためのプラットフォームです。
お使いのPCにインストールされているか以下の手順で確認します。

1. ターミナル（コマンドプロンプト）を開きます。
2. 次のコマンドを入力して実行します。

```bash
node -v
```

もしバージョン番号（例: `v18.18.2`）が表示されたら、Node.jsはすでにインストールされています。
表示されない場合は、[Node.jsの公式ウェブサイト](https://nodejs.org/)からインストールしてください。

### プロジェクトのセットアップ

プロジェクトフォルダに移動し、必要なnpmパッケージ（プログラミングのコードを再利用しやすくするためのパッケージ）をインストールします。
以下のコマンドをターミナルで実行してください。

```bash
npm install
```

### 開発サーバーの起動

インストールが完了したら、開発サーバーを起動して、ブラウザでプレビューします。
以下のコマンドを実行します。

```bash
npm run develop
```

成功すると、`http://localhost:8000/`と表示されます。
このURLをブラウザに貼り付けてアクセスすると、テンプレートのサイトが表示されます。

### 404ページの確認

サイトを公開する前に、404ページ（存在しないページにアクセスしたときに表示されるページ）が正しく設定されているか確認します。
以下のコマンドで、本番環境用のビルドとサーバーを起動します。

```bash
npm run build
npm run serve
```

次に、`http://localhost:9000/`にアクセスし、存在しないページ（例えば`http://localhost:9000/thispagedoesnotexist`）を試してみてください。

## 編集とカスタマイズ

- **ページの編集:** サイトのページは`src/pages`ディレクトリ内にあります。このディレクトリを開いて、ファイルを編集することで、サイトのコンテンツを変更できます。
- **コンポーネントの追加:** サイトの見た目をカスタマイズするために、`src/components`ディレクトリに新しいReactコンポーネントを追加することができます。
